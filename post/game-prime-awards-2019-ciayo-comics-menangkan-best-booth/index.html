Game Prime Awards 2019: CIAYO Comics Menangkan Best Booth

Game Prime Awards selalu jadi perhelatan yang ditunggu-tunggu dalam BEKRAF Game Prime. Segmen kecil ini merupakan bentuk apresiasi kepada para partisipan Game Prime atas prestasinya selama satu tahun terakhir maupun selama acara berlangsung.

Di tahun 2019 ini, Game Prime Awards kembali diselenggarakan di acara BEKRAF Game Prime 2019. Meskipun kali ini hanya mengangkat empat nominasi saja, prestise setiap penghargaan yang diumumkan tanggal 14 Juli lalu ini tidak boleh diabaikan begitu saja. Apa saja nominasinya dan siapa saja pemenangnya? Langsung saja disimak di bawah ini!

Nominasi dan Pemenang BEKRAF Game Prime Awards 2019

Citizen Choice Award: Jurit Malam (Gambir Studio)

Formula untuk memenangkan Citizen Choice Award adalah dengan membuat game yang sangat relatable dan mudah disukai oleh banyak orang. Melalui polling dalam acara Game Prime 2019, game Jurit Malam berhasil menjadi game favorit pilihan pengunjung.

Meskipun Jurit Malam menampilkan gameplay tipikal banyak game horor di pasaran, X-factor yang menjadikan game ini menonjol adalah karena penampilan sosok model dan cosplayer Lola Zieta sebagai playable character dalam game tersebut. Apa kamu sendiri tertarik memainkan Lola Zieta dalam game horor ini?

Board Game of the Year: Jalan-Jalan (Chiveus)

Board game Jalan-Jalan buatan Chiveus berhasil memenangkan penghargaan Board Game of the Year. Tentunya ada alasan kenapa board game ini berhasil menarik perhatian dan menjadi yang terbaik.

Kamu dan teman-teman kamu harus memilih akan pergi berlibur ke mana. Dengan puluhan jenis tempat wisata, tentunya tidak semua bisa didatangi. Voting suara terbanyak akan menentukan tujuan selanjutnya, namun meskipun pilihanmu kalah suara kamu bisa mempengaruhi hasilnya dengan berbagai macam trik dan power up.

Game of the Year: Rage in Peace (Rolling Glory)

Banyak game menjadikan kematian sebagai titik game over. Namun, Rage in Peace menjadikan kematian sebagai tujuan yang harus dicapai. Meskipun game platformer ini memungkinkan karaktermu untuk mati dengan segala cara yang sadis, tujuan akhir game ini adalah menuntun sang main character Timmy pulang ke rumah untuk beristirahat dengan tenang.

Tidak hanya menyajikan gameplay yang adiktif, Rage in Peace juga punya cerita kontemplatif serta soundtrack bergaya indie dari ikkubaru, lightcraft, Peonies, Sajama Cut, dan band-band indie lainnya. Dengan presentasi lengkap tersebut, tak heran Rage in Peace buatan Rolling Glory mendapatkan gelar Game of the Year.

Best Booth: CIAYO Mart (CIAYO Comics)

Last but not least, CIAYO Comics mendapatkan penghargaan Best Booth di BEKRAF Game Prime 2019! CIAYO Comics hadir dengan booth besar yang luas dan keren. Mengusung konsep minimarket, booth CIAYO Mart menjual berbagai pernak-pernik CIAYO Comics serta merchandise buatan para author komik.

Para pengunjung bisa datang ke booth CIAYO Mart untuk belanja, makan es krim gratis, atau foto-foto di belakang rak sirup yang Instagrammable banget!

Para pemenang BEKRAF Game Prime Awards 2019 mendapatkan plakat sebagai memorabilia. Selain itu, pemenang Citizen Choice Award dan Game of the Year akan diterbangkan ke Singapura untuk menghadiri GameStart 2019 pada tanggal 12-13 Oktober 2019!

Selamat untuk para pemenang, dan teruslah berkarya! Sampai jumpa di Game Prime Awards tahun selanjutnya!
