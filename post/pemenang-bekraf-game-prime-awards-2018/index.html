Ini Dia Para Pemenang BEKRAF Game Prime Awards 2018!

BEKRAF Game Prime 2018 telah usai. Para developer game lokal sukses memamerkan game terbaik buatan mereka. Namun, siapakah yang terbaik di antara semuanya? BEKRAF Game Prime Awards 2018 merupakan bentuk apresiasi terhadap para developer dan publisher game Indonesia yang telah berjuang memajukan industri game lokal hingga harum namanya di dalam negeri maupun panggung internasional.

Total 50 developer game berpartisipasi dalam Game Prime 2018, dan beberapa dari mereka berhasil mencatatkan namanya sebagai pemenang penghargaan Game Prime Awards 2018. Siapa saja yang berhasil meraih penghargaan Game Prime Awards 2018?

Most Innovative Gameplay: King’s Ploy

Sumber: Game Prime

Penghargaan Most Innovative Gameplay berhasil didapatkan game King’s Ploy dari developer Mintsphere. Game puzzle ini mengangkat konsep permainan catur dengan petak yang lebih kecil dan rangkaian bidak catur yang berbeda-beda di setiap misinya. Tak hanya menyajikan permainan yang menarik dan unik, kamu juga bisa belajar konsep permainan catur dengan lebih menyenangkan.

Most Promising Game: Forged of Blood

Sumber: KAORI Nusantara

Game strategi Forged of Blood karya Critical Forge berhasil meraih penghargaan Most Promising Game. Forged of Blood bisa jadi salah satu breakthrough untuk industri game Indonesia, karena menyajikan permainan strategi turn based yang solid ala XCOM, serta presentasi visual 3D yang sangat memukau.

Best Tabletop Game: Candrageni

Sumber: Game Prime

Kompas berhasil memenangkan penghargaan Best Tabletop Game lewat board game mereka, Candrageni. Dalam board game yang bersetting di kerajaan Mataram ini, pemain harus membangun candi sekaligus berdagang dengan negara dan kerajaan lainnya.

Best Booth: Own Games

Sumber: Game Prime

Own Games datang ke Game Prime 2018 untuk mengenalkan game terbaru mereka, Own Coffee Shop. Untuk menarik perhatian, mereka membangun kedai kopi betulan! Di sana, pengunjung bisa membeli kopi dan jajanan lainnya. Dengan konsep unik ini, Own Games berhasil memenangkan penghargaan Best Booth.

Best Publisher: Toge Productions

Sumber: Game Prime

Meski bukan yang terbesar, Toge Productions menjadi salah satu publisher game Indonesia yang cukup prolific dan aktif merilis game-game keren. Tak hanya menerbitkan Ultra Space Battle Brawl di Nintendo Switch dan My Lovely Daughter di Steam, Toge Productions juga membantu pendanaan Game Changer Studios sehingga mereka mampu merampungkan game My Lovely Daughter. Atas sumbangsihnya, tak heran Toge mampu memenangkan penghargaan Best Publisher.

Citizen Choice Award: Overgod


Selama Game Prime 2018, pengunjung bisa memilih satu booth yang menurut mereka punya game terbaik. Bekerja sama dengan Mfun, penghargaan Citizen Choice Award jatuh kepada game Overgod milik studio game RedRain.

Game of the Year: Ultra Space Battle Brawl

Sumber: Nintendo

Penghargaan tertinggi Game Prime 2018 jatuh kepada Ultra Space Battle Brawl yang menjadi game of the year. Game produksi Mojiken Studio ini jadi salah satu party game paling solid karena permainannya yang sederhana, namun sangat fast paced. Proses pembuatan USBB jadi cerita yang sangat menarik, karena mempopulerkan Nintendo Switch sebagai salah satu konsol game yang jadi target incaran para developer game Indonesia.

Game Industry Icon: Hari Sungkari

Sumber: Game Prime

Last but not least, ada penghargaan Game Industry Icon yang diberikan kepada bapak Hari Sungkari. Beliau adalah deputi infrastruktur dari BEKRAF yang senantiasa mendukung para developer Indonesia dari balik layar.

Itu dia para pemenang BEKRAF Game Prime Awards 2018. Apa game jagoan kamu jadi pemenangnya? Semoga penghargaan ini bisa menjadi bentuk penyemangat bagi para developer game lokal untuk terus berkarya dan menciptakan game yang terbaik.
