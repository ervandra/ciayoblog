Anime Fireworks: Sebuah Kisah Tentang Perbedaan Perspektif (Should We See from the Side or the Bottom)?

Menurut kamu kembang api itu bentuknya datar atau bundar? hmm, jadi penasaran kan? Sebelumnya bro sis pasti ga pernah bertanya-tanya tentang hal ini. Nah, Untuk tau jawabannya, kita harus banget nih nungguin anime movie yang sebentar lagi bakal tayang. Anime Fireworks, Should We See It from The Side or The Bottom? (atau Uchiage Hanabi, Uekara Miruka? Shitakara Miruka?) adalah adaptasi dari TV drama karya Shunji Iwai di tahun 1993. Film ini diarahkan oleh Nobuyuki Takeuchi, yang merupakan penata visual dan production designer dari serial Shaft Monogatari. Selain itu, Akiyuki Shinbo sebagai general director.

(Sumber: wikipedia)

Perbedaan Anime Fireworks Dengan Versi Drama TV

Jika disesuaikan dengan alur cerita dari drama TV-nya, anime Fireworks/Uchiage Hanabi ini diawali dengan perdebatan antara Norimichi dan Yusuke dan teman sekolahnya di pinggiran kota Jepang mengenai, apakah kembang api itu sebenarnya ‘datar’ atau ‘bundar’ ketika meledak di langit. Sementara itu, Nazuna punya hal lain untuk dipikirkan karena orang tuanya yang akan berpisah. Karena perceraian tersebut, rencananya Nazuna akan pindah setelah liburan akhir musim panas. Ketika Norimichi dan Yusuke sedang melakukan lomba renang, Nazuna memutuskan untuk mengajak pemenang dari lomba itu untuk kabur dari rumah dengannya. Mereka juga berencana untuk menonton kembang api dari menara mercusuar supaya tau jawaban dari perdebatan di sekolah.

(Sumber: pinterest)

Hanabi atau kembang api ini pasti sering banget deh kita lihat di tiap anime Jepang. Kembang api biasanya identik dengan festival, atau perayaan besar tertentu yang lekat banget dengan budaya Jepang. Hanabi sendiri berasal dari kata ‘Hana’ yang berarti bunga dan ‘Bi’ yang berarti api. Tradisi kembang api ini sudah dimulai sejak tahun 1732, ketika Jepang masih mengalami krisis ekonomi, wabah penyakit, dan kelaparan. Orang jepang menggunakan kembang api sebagai media untuk menghibur mereka mengenai keadaan saat itu. Kembang api digambarkan sebagai sebuah simbol yang mengingatkan kita tentang harapan dan kebahagiaan. Lalu sejak itulah,  kembang api sering dipakai saat festival musim panas dan hampir di setiap perayaan.
