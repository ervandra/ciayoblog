Game Smartphone Naruto x Boruto: Ninja Voltage Kini Tersedia Untuk Seluruh Region

Game Naruto sangat banyak di luar sana. Sejak era PlayStation One hingga PlayStation 4 seperti sekarang, ada berbagai jenis game yang menggunakan tema dunia Naruto. Mulai dari genre standar seperti fighting, team-battle ninja games, sampai genre “defend the fortress” seperti yang banyak di adopsi game smartphone belakangan ini. Video game Naruto punya hampir semua genre permainan.

Ini game Naruto untuk PS4. Sumber gambar: store.playstation.com

Yang terkini dari game smartphone bertema Naruto adalah game berjudul “Naruto x Boruto: Ninja Voltage”. Game ini di-publish oleh Bandai Namco Entertainment, yang sudah pernah menerbitkan game-game Naruto sebelumnya.

Game smartphone bertema Tower Defense. Sumber gambar: youtube.com

Naruto x Boruto: Ninja Voltage awalnya hanya tersedia untuk region Jepang mulai pertengahan November 2017 di dua platform mobile terbesar saat ini: Google Android dan Apple iOS. Seminggu kemudian, region lain baru kebagian (termasuk Asia Tenggara tentunya).

Versi berbahasa internasional sudah dirilis. Sumber gambar: youtube.com

Seperti game “tower defense” lain di smartphone (atau bahkan di PC sekalipun), game Ninja Voltage ini dapat di-download secara gratis untuk dimainkan. Pemain akan dapat melakukan micro-transaction di dalam game ini, seperti misalnya membeli aksesoris pendukung permainan.

Konsep Freemium seperti ini sudah jadi praktek umum di game smartphone. Bahkan publisher video game sebesar Nintendo sekalipun juga ikutan bermain di area “gratisan tapi bayar” ini. Game gratis di smartphone jarang (atau malah tidak ada?) yang benar-benar “gratis” karena di satu titik, pemain akan harus mengeluarkan uang beneran (atau koin dalam game, yang dibeli pakai uang beneran) untuk dapat terus bermain. Atau untuk dapat melewati satu hambatan di dalam game, yang nyaris mustahil untuk dilalui tanpa melakukan micro-transaction di dalam game gratisan tadi.

Tampilan status karakter di game Ninja Voltage. Sumber gambar: facebook.com/NinjaVoltage

Game Naruto x Boruto: Ninja Voltage ini akan mengalami hal yang sama. Micro-transaction.

Karena game ini bertema “tower defense” maka pemain game smartphone yang sudah terbiasa dengan game seperti Mobile Legends (atau bahkan game sejuta umat “DotA” di PC) tidak akan begitu kesulitan mengadaptasi gameplay yang ditawarkan. Informasi di website resmi game ini ( http://ninjavoltage.bn-ent.net/ ) juga menjelaskan sekilas seperti apa gameplay yang di tawarkan oleh Ninja Voltage ini.

  • Attack Fortresses/Serang Benteng Yang Ada

Dengan menggunakan karakter-karakter familiar dari serial Naruto + Boruto yang sudah kita kenali dengan baik, kita akan menyerang benteng pertahanan lawan. Lewati rintangan yang ada (termasuk jebakan dan para ninja penjaga), jelajahi daerah kekuasaan musuh dan kuasai sumber daya mereka.

Attack! Sumber gambar: ninjavoltage.bn-ent.net

  • Defend Fortress/Pertahankan Benteng Anda

Sebaliknya, kita akan membangun benteng untuk melindungi desa serta sumber daya yang dimiliki. Perkuat para ninja penjaga benteng, bangun berbagai jebakan untuk mencegah musuh datang masuk mencuri sumber daya, dan pastikan benteng tetap berdiri tegak dalam situasi kemenangan.

Defend! Sumber gambar: ninjavoltage.bn-ent.net

  • Multiplayer Mode/Mode Permainan Multiplayer

Tidak harus sendirian dalam menyerang ataupun mempertahankan benteng kita! Lakukan kerjasama dengan teman-teman pemain Ninja Voltage lainnya. Kalahkan musuh dengan kerjasama melalui mode multiplayer. Ingat, bersatu kita teguh!

Multiplayer Mode yang seru. Sumber gambar: ninjavoltage.bn-ent.net

  • Enjoy The Cinematic Ninja Action/Nikmati Aksi Ala Ninja Naruto

Ambil bagian dalam berbagai aksi ninjutsu dari karakter-karakter populer universe Naruto x Boruto di game seru ini. Berbagai jurus, jutsu, ilmu dan cutscene disajikan secara sinematik sehingga akan terasa lebih hidup dan seperti anime Naruto x Boruto yang kita kenal dan sukai selama ini. Pastinya akan seru banget.

Serasa nonton anime-nya juga. Sumber gambar: ninjavoltage.bn-ent.net

Terdapat pula sistem Battle Ranking dimana kita dapat melihat sehebat apa tingkatan kita di antara pemain Naruto x Boruto: Ninja Voltage lainnya yang tersebar di seluruh dunia.
