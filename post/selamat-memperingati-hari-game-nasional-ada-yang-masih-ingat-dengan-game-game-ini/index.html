Selamat Memperingati Hari Game Nasional! Ada yang Masih Ingat Dengan Game-Game Ini?

Di tengah berkembangnya industri game di Indonesia beberapa tahun belakangan ini, membuat Asosiasi Game Indonesia, tepatnya tahun lalu, menetapkan tanggal 8 Agustus sebagai Hari Game Nasional. Hal ini bertujuan untuk memberikan wadah bagi para pelaku industri game, bahkan pemain untuk mendukung dan berpartisipasi dalam mendorong kemajuan industri game Indonesia.

Nah, tepatnya hari ini tanggal 8 Agustus 2017, dalam rangka memperingati Hari Game Nasional, yuk kita telaah sedikit mengenaiÂ sejarah perkembangan game online di Indonesia. Siap-siap bernostalgia yuk!

Nexia Online

Nexia Online (Sumber: nexia online wiki)

Taukah kamu bahwa game online pertama yang masuk ke Indonesia adalah Nexia? Ya, game online RPG ini muncul di bulan Maret 2001. Dengan grafis 2d , Nexia sudah bisa dimainkan di komputer Pentium 2. Dengan empat karakter utama yaitu mage, roque, warrior dan poet, game ini punya dunia jelajah yang luas, Sayangnya, game ini sudah gantung sepatu pada tahun 2004 dan sepertinya belum ada tanda-tanda game ini akan dihidupkan kembali.

RF Online

RF Online (Sumber: free2play)

Wah, bagi kamu sesepuh gamers Indonesia pasti uda ga asing lagi sama game ini. Game 3D yang mengharuskan kamu men-upgradeÂ komputer ini bisa dikatakan sebagai era revolusi game di Indonesia. Yang istimewa dari masa ini adalah, pemain sudah mulai beralih dari koneksi internet dial-up, ke internet langganan. Para gamers di zaman ini juga menjadi lebih serius bermain, karena mereka sudah mulai membidik kekuasaan dan kemenangan. Tak hanya pemain, pelaku industri dunia game pun mulai bermunculan.

Ayo Dance

Ayo Dance (Sumber: youtube)

Di masa Ayo Dance ini lah gamers perempuan mulai menunjukkan aksinya. Game ini tak hanya ramai oleh para laki-laki, namun ada banyak juga perempuan yang ikut bermain. Dengan konsep couple dan ‘styling’ yang ada di dalam game-nya, pacaran online lewat game ini pun mulai bermunculan. Apalagi kita diperbolehkan untuk mencari couple di game ini. Bahkan salah satu teman saya, berkenalan dengan pacarnya lewat Ayo Dance loh! Hubungannya pun masih bertahan hingga sekarang. Wow~

Game ini memang dikatakan sangat adiktif. Di masa inilah, warnet-warnet pun mulai jaya, apalagi Ayo Dance mempunyai mi-cash, yaitu voucher dimana Â pemain bisa membeli sejumlah item di dalam game dengan uang asli.

Point Blank

Point Blank (Sumber: gaming.youtube)

Game online dengan jutaan pemain dan fanbase di seluruh dunia ini memang pantas dijadikan salah satu game terpopuler di Indonesia. Bahkan dengan kesuksesan Point Blank, banyak perusahaan game lain yang berbondong-bondong membuat game-game sejenis seperti CrossFire, WarRock dan X-SHOT. Kemunculan PointBlank sekaligus menjadi pencetus turnamen-turnamen game online di Indonesia.

Ragnarok Online (RO)

Ragnarok. (Sumber: encyclopedia.gamia)

Game ini juga terbilang berhasil membumi-hanguskan dompet para pemainnya. Awal mula game ini dikabarkan akan masuk ke Indonesia, gamers Indonesia sudah sangat antusias menantikan kehadirannya. Game yang booming sejak tahun 2003 ini, mempunyai gameplay yang menarik dan adiktif. Sehingga tak heran, game ini berhasil sukses di negara belahan Asia dan Eropa.

Bisa dikatakan setelah masa game-game diatas, selanjutnya yang berjaya adalah mobile game seperti Clash of Clans, Clash Royale dan lain-lain. Bahkan saat ini game online tidak hanya dijadikan untuk menghabiskan waktu dan sebagai aktivitas menyenangkan lagi. Game online mulai dimanfaatkan sebagai ladang bisnis bagi para pelakunya.
