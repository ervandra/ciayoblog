Mengenal Mario Kart: Game Balap Legendaris Nintendo yang Akan Muncul di Smartphone

Semakin hari sepertinya Nintendo semakin serius dengan platfom mobile / smartphone. Setelah game smartphone perdana mereka yang fenomenal (walaupun tidak selaris yang di prediksi para analis) “Mario Run” , Nintendo terus menggebrak pasar game mobile. Mario Run disusul game staple Nintendo lainnya “Animal Crossing: Pocket Camp”  yang memiliki model marketing yang sama: Freemium. Walau game-nya bisa di download dengan gratis tapi gamers wajib mengeluarkan sejumlah dana/uang untuk membeli full version / unlock features dari game tersebut karena yang diberikan gratis oleh Nintendo pada prinsipnya tak lebih dari sekedar “demo game” untuk dicoba calon pembeli.

Dilaporkan kalau Nintendo memiliki rencana untuk merilis 2-3 mobile games per tahun. Bahkan Zelda termasuk di dalam rencana ini! Bayangkan, bro sis… Game signature Nintendo seperti Zelda sebentar lagi akan ada versi smartphone! … Tapi sebenarnya saya tidak bisa membayangkan game / bermain game Zelda di smartphone. Dengan kontrol d-pad dan tombol saja saya kesulitan menguasai Zelda. Gimana lagi kalau hanya diberi kontrol layar sentuh khas smartphone?

Ini game Zelda untuk 3DS. Mungkin Zelda smartphone akan seperti ini juga? Sumber gambar: ebay.com

Anyway! Mario Kart. Game apa ini?

Mario Kart adalah game ekslusif Nintendo. Sumber gambar: kotaku.com.au

Ini game yang legend banget dari Nintendo. Termasuk game spin-off dari game utama Super Mario Bros, Mario Kart memiliki fans fanatik yang tersebar di seluruh dunia. Mario Kart pada dasarnya adalah sebuah game balapan go-kart. Dan karena berupa game balapan, tentu tujuan dari game ini adalah untuk menjadi yang pertama kali mencapai garis finish ‘kan? Nah… serunya di game Mario Kart, para pemain diperbolehkan “bermain curang” sepanjang balapan. Seperti menimpuki lawan dengan tempurung kura-kura, kulit pisang (yang bisa menyebabkan terpelintir keluar dari lintasan) hingga memakai Turbo Boost yang dapat membantu pemain melesat kencang melewati lawan-lawannya.

Timpuk lawan dengan kulit pisang dan dia akan terpeleset keluar lintasan. Sumber gambar: nintendolife.com

Serunya ‘balapan rasa perang’ ini menjadikan game Mario Kart bukan saja soal adu cepat seperti halnya game balapan, tapi juga adu strategi agar lolos dari jebakan dan timpukan lawan.

Game pertama dari franchise ini adalah “Super Mario Kart” untuk console Super Nintendo (SNES) dan Super Famicom (SuFami) di versi Jepang. Super Mario Kart diluncurkan ke pasar tahun 1992. Di produseri oleh ‘bapaknya’ Mario, Shigeru Miyamoto, game ini sukses besar dengan terjual lebih dari sembilan juta unit dan di-port ke mesin-mesin Nintendo selanjutnya pasca SNES/SuFami. Dengan konsep game balapan go-kart yang dipadu kemampuan menghambat laju lawan menggunakan berbagai benda dan taktik menjadikan Super Mario Kart sebuah legenda klasik game Super Nintendo. Gelar / termasuk dalam golongan “Best Game of All Time” juga didapatkan Super Mario Kart dari berbagai media gaming. Bahkan Guiness World Records tak ragu menggelari Super Mario Kart sebagai “Game Console Paling Top”.

Game dengan berbagai rekor dan penghargaan. Sumber gambar: lukiegames.com

Terlepas dari segala puja-puji yang didapatkan Super Mario Kart, game ini memang secara langsung menjadi inspirasi banyak game balap go-kart berikutnya, termasuk di console modern seperti PlayStation. Pernah main “Crash Team Racing”? “Chocobo Racing”? Atau seabreg-abreg game balap go-kart dimana kita bisa menghajar lawan agar kita jadi yang terdepan? Semua dari mereka merupakan derivat dan terinspirasi dari game Super Mario Kart-nya Super Nintendo.

Gamers PS1 pasti kenal game ini! Sumber gambar: gearnuke.com

Selepas Super Mario Kart, sekuel dan spin-off dari game itu bermunculan di berbagai platform Nintendo, baik itu di console maupun handheld. Bahkan arcade! Karena Nintendo juga membuat game balap Mario Kart yang ditujukan untuk pasar arcade center. Tercatat ada empat game arcade bertema Mario Kart sejak tahun 2005 hingga 2017 kemarin, dan semuanya merupakan kerjasama Nintendo dengan Namco/Bandai Namco. Konsep game tetap sama seperti game Mario Kart lainnya, namun tentu saja dengan tampilan grafis mentereng khas mesin arcade. Saya pernah melihat mesin arcade Mario Kart ini di lobby bioskop terkemuka di kota Medan, dan saya 100% yakin kalau di kota-kota besar lainnya di Indonesia juga punya mesin ini.

Bermain Mario Kart dengan rasa arcade. Sumber gambar: bandainamco-am.com

Konfirmasi game terbaru Mario Kart untuk platform smartphone dilakukan Nintendo via akun Twitter resmi mereka. Tidak banyak informasi yang diberikan di tweet tersebut selain judul game-nya adalah “Mario Kart Tour”. Juga tidak ada tanggal rilis definitif di tweet tadi selain kalimat “akan dirilis sebelum akhir tahun fiskal Maret 2019”. Masih lumayan lama ya, bro sis?

Tweet resmi Nintendo soal game Mario Kart mobile. Sumber gambar: twitter.com/NintendoAmerica
