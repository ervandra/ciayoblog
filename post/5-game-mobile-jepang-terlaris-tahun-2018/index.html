5 Game Mobile Jepang Terlaris Tahun 2018

Mendengar kata mobage atau game mobile Jepang kalian pasti langsung terbayang dengan gacha. Gacha menjadi salah satu ciri khas yang paling dominan dalam game smartphone saat ini. Tidak heran kalau banyak sekali player mobage rela mengeluarkan uang demi mendapatkan salah satu fitur dalam game, mulai dari kostum, item, senjata, dan yang pasti waifu/husbando.

Menjelang penutupan tahun 2018 ini, salah satu situs game Jepang yaitu Social Game Info mengumumkan 5 mobage yang memiliki penghasilan tertinggi pada tahun 2018 ini. Apa saja game mobile Jepang terlaris tahun 2018 ini? Apakah mobage favorit kalian termasuk didalamnya?  Mari kita simak bersama-sama.

Inilah 5 Game Mobile Jepang Terlaris Tahun 2018

5. Kouya Koudou: Knives Out (44.8 Milyar Yen)

Kolaborasi game Kouya Koudou: Knives Out dengan anime Attack on Titan. (Sumber: Reddit)

Urutan ke-5 game mobile Jepang terlaris tahun 2018 dikuasai oleh sebuah game Battle Royale Shooter, yaitu Kouya Koudou: Knives Out. Game yang berasal dari developer NetEase sangat populer di Jepang. Percaya atau tidak, game ini pernah mengadakan event collab dengan Shingeki no Kyojin/Attack on Titan. Selain itu beberapa Virtual Youtuber seperti Kizuna Ai dan Siro bermain game Kouya Koudou: Knives Out disalah satu video di channel mereka. Saya sendiri kaget melihat game ini ternyata populer di Jepang dan menghasilkan penghasilan yang besar dibandingkan dengan game lain yang memiliki gameplay sama dengan Kouya Koudou: Knives Out. Game ini tercatat telah menghasilkan penghasilan pada tahun 2018 sekitar 44.8 milyar Yen atau sekitar hampir 5.8 triliun Rupiah.

4. Dragon Ball Z: Dokkan Battle (56.1 Milyar Yen)

Inilah satu dari sekian banyak adaptasi game Dragon Ball, termasuk versi game gacha. (Sumber: Gamerpros)

Siapa sih yang tidak kenal dengan Dragon Ball? Karya manga dari Akira Toriyama ini sangat populer, mulai dari anime, game console-handheld, taman hiburan dan hingga akhirnya dibuat juga game khusus untuk gacha smartphone. Dragon Ball Z: Dokkan Battle berhasil masuk dalam urutan ke-4 sebagai game mobile Jepang terlaris tahun 2018. Game ini memiliki banyak karakter dari seri Dragon Ball, mulai dari Goku,Vegeta, Frieza, Krilin, Android 18 dan masih banyak lagi. Tentu saja karakter-karakter ini bisa kalian dapatkan melalui gacha. Dragon Ball Z: Dokkan Battle berhasil meraih penghasilan pada tahun 2018 sebesar 56.1 milyar Yen atau sekitar 7.2 triliun Rupiah.

3. Puzzle & Dragon (59.4 Milyar Yen)

Karakter Puzzle & Dragon yang bisa didapatkan melalui gacha. (Sumber: Markinternational)

Puzzle & Dragon adalah sebuah game smartphone dengan mencampurkan genre antara puzzle, RPG dan strategi. Game yang dibuat oleh developer GungHo Online Entertainment tidak kalah populernya dengan Dragon Ball Z: Dokkan Battle yang berhasil menduduki urutan ke-3 sebagai game mobile Jepang terlaris tahun 2018. Selain itu, kepopuleran game ini dapat dilihat dari sudah berapa banyak collab dengan franchise lain seperti anime, manga, game dan lain lain. Pencapaian game Puzzle & Dragon berhasil meraih penghasilan pada tahun 2018 sebesar 59.4 milyar Yen atau sekitar 7.6 triliun Rupiah.

2. Fate/Grand Order (105 Milyar Yen)

FGO selalu menjadi pusat perhatian dunia mobage. (Sumber: Tokyo Otaku Mode)

Saya tidak heran jika game Fate/Grand Order atau disingkat FGO masuk dalam kategori ini. Game dari developer Delightworks atau lebih dikenal dengan DW merilis game berdasarkan salah satu franchise TYPE-MOON berhasil menguasai urutan ke-2 sebagai game mobile Jepang terlaris tahun 2018. Banyaknya godaan waifu/husbando pada tahun ini membuat para para pemain kehilangan akal mereka. Mulai dari kemunculan servant dengan class baru Foreigner bernama Hokusai, bangkitnya team meta quick oleh servant Scathach=Skadi, husbando tampan dengan kacamata bernama Sigurd, rate up summer dan masih banyak lagi. Saat ini, FGO telah meraih penghasilan sebesar 105 milyar Yen atau sekitar 13.5 triliun Rupiah.

1. Monster Strike (107.6 Milyar Yen)


Monster Strike menjadi game mobile Jepang terlaris tahun 2018. (Sumber: Tech in Asia)

Pada awalnya saya mengira posisi nomor satu game mobile Jepang terlaris tahun 2018 akan dikuasai oleh game Granblue Fantasy, Idolm@ster atau BanG Dream. Ternyata nomor satu dikuasai oleh game dari developer Mixi yaitu Monster Strike. Game yang telah berdiri lama sejak tahun 2013 dengan sistem gameplay mirip sebuah billiards sampai saat ini masih sangat populer di Jepang. Selain mobage, Monster Strike melebarkan sayapnya dengan merilis game di Nintendo 3DS dan sebuah adaptasi anime beserta movie. Dengan kepopuleran seperti ini, developer Mixi berhasil meraih penghasilan tertinggi dengan mobage Monster Strike mereka, yaitu mencapai 107.6 milyar Yen atau sekitar 13.9 triliun Rupiah.

Terik terus, gan! (Sumber: Fate/Grand Order)
